# Traffic Control Script

Traffic Control Script is a FiveM traffic management resource pack built for servers that want cleaner, easier in-game traffic control without relying on constant config edits or server restarts.

This pack includes **two versions**:

## Full Version
The full version is designed for servers that want the complete traffic management framework.

### Features
- Global traffic presets
- Custom density sliders
- Local traffic control scenes
- Access management
- Unified traffic management menu

## Lite Version
The lite version is designed for servers that only want global traffic control without the local scene system.

### Features
- Global traffic presets
- Custom density sliders
- Access management
- Same overall control flow as the full version
- No local traffic scene system

## Main Difference
- **Full** includes local scene-based traffic control
- **Lite** keeps only the global traffic control system
